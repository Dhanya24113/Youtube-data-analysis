📊 YouTube Data Analysis (Python)

📌 Project Overview

This project focuses on analyzing YouTube trending video data to extract meaningful insights using Python.
It involves data cleaning, exploration, and visualization to understand patterns in user engagement such as views, likes, and categories.

---

🎯 Objectives

- Analyze relationships between views, likes, and dislikes
- Identify top-performing video categories
- Discover trending channels and content patterns
- Visualize data for better business insights

---

🛠️ Tech Stack

- Python
- Pandas (Data Cleaning & Analysis)
- NumPy (Numerical Operations)
- Matplotlib & Seaborn (Data Visualization)

---

📂 Dataset

- Source: YouTube Trending Dataset (CSV file)
- Contains information about:
  - Video title
  - Category
  - Views, Likes, Dislikes
  - Channel name

---

📊 Key Insights

- Strong correlation observed between views and likes
- Certain categories dominate trending content
- Top channels consistently appear in trending lists
- Engagement patterns vary across categories

---

📈 Visualizations Included

- 📊 Bar Chart → Video category distribution
- 📉 Scatter Plot → Views vs Likes
- 📉 Scatter Plot → Likes vs Dislikes
- 🔥 Heatmap → Correlation between features
- 🥧 Pie Chart → Category share
- 📊 Bar Chart → Top channels
